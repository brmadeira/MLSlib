MLS Surface Library
-------------------

1) Author Information
Bruno Eduardo Madeira
madeira@ime.eb.br
19 Feb 2008

2) About
This is a library written in C++, that implements the algorithm presented 
in the paper “Defining Point-Set Surfaces”, by Nina Amenta and Yong Joo Kil,
in the Siggraph 2004. This algorithm allow us to sample points over a surface
defined by a point cloud.

3) Requirementes
GSL Library

4) To Install
Type make.

5) To test
Execute the program /demo/mlstest.

6) To Use
Read the documentation in /doc
